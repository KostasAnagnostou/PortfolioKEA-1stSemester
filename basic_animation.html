Basic Animation

This was the 3rd Theme of the 1st Semester at KEA - Copenhagen School of Design and Technology.

It contained 4 modules (JavaScript part 1, Character Design & storyline, JavaScript part 2, Group assignment, exhibition, test, feedback ). We did both individual and group projects.

The theme ended out with a small set of interactive animations, an interactive story with focus on the story, a style, character design, animations and interactivity designed for a specific target group. The projects documented and tested on different stages of development.

We learned & practiced with:
Animation principles, drawing of visual universe and characters in illustrator, hand drawing, sketching, character sheets, psychological profiles - personas collage, dramaturgical elements, the narrative curve, SVG files, Audio Sound, Advanced CSS Selectors, Transitions, Keyframe Animations, Pseudo code, basic concepts in JavaScript (built-in & custom Functions, Events & EventListeners, querySelector, classList, Variables, setTimeout / setInterval, Controlling audio), Sequence diagrams and more...

I started with a Simple Animation using JS and CSS Animations, and then I created a Story with 3 Characters which I designed them in Illustrator. After that, we had only group projects. Firstly, we created an Interactive Animation from one member's Storyline (we were 3 students in the Group).

Subsequently, we created a very basic Calculator and some simple SVG Animations, and then followed the final Interactive Project (in a Group of 7 Students):

We were given a Style (Surrealism in our case) & a Piece of music: "Jimi Hendrix: ...And the Gods Made Love", and we created an Interactive Animation Story based on Pinocchio. We designed "our" Pinocchio character, and all the graphics (backgrounds, sprites, etc.), the personas collage, we created the storyboard, we did User Tests, and then we coded the Interactive Animation Story with HTML5, CSS and JavaScript.

We used many SVG Vectors for higher interactivity, along with png, jpg images and a small moving gif image, which we created on After Effects. Lastly, we did a screencast documentation (I did the coding part on the screencast).
