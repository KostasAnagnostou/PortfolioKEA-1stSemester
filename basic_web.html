Basic Web

This was the First Theme of the 1st Semester at KEA - Copenhagen School of Design and Technology.

I was given a style (Vintage Style in my case) and I created a basic Responsive Website using raw Grid CSS.

I designed & created the Paper & the Interactive Prototypes, the Logo (using Illustrator), the Style Tile (typography, colors and more...), applying the Design Principles (Symmetrical / Assymetrical Composition, Gestalt Principles, etc.) that I learned during the Basic Web Theme.
