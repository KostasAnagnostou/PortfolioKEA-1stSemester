# PortfolioKEA-1stSemester
This is a simple web portfolio with all my projects at 1st Semester at KEA - Copenhagen School Of Design &amp; Technology. The creation of web portfolio is the final part of the 1st Semester.  

Requirements for the portfolio website<br>
● The portfolio website must be "hand coded" using HTML, CSS, and JavaScript. I.e. you<br>
are NOT allowed to use CMS or frameworks, such as Wordpress or Bootstrap.<br>
● All pages of your portfolio website must validate for both HTML and CSS<br>
● The portfolio website must be responsive.<br>
● Your portfolio website should at least have the following pages:<br>
○ A homepage<br>
○ A CV / “About me” page<br>
○ Your portfolio — one or more pages presenting your 1st-semester projects<br>
