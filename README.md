# PortfolioKEA-1stSemester
This is a simple web portfolio with all my projects at 1st Semester at KEA - Copenhagen School Of Design &amp; Technology. The creation of web portfolio is the final part of the 1st Semester.  

Requirements for the portfolio website
● The portfolio website must be "hand coded" using HTML, CSS, and JavaScript. I.e. you
are NOT allowed to use CMS or frameworks, such as Wordpress or Bootstrap.
● All pages of your portfolio website must validate for both HTML and CSS
● The portfolio website must be responsive.
● Your portfolio website should at least have the following pages:
○ A homepage
○ A CV / “About me” page
○ Your portfolio — one or more pages presenting your 1st-semester projects
